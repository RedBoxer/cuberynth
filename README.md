# cuberynth

В папке С# решение первый задачи. Поддерживает целые числа, сложение, вычитание, умножение, скобки, деление.

Всё остальное это решение четвёртой задачи. Проект генерирует игрушку куб лабиринт. Размер генерируемого куба можно менять поменяв константу labSize в GameCube и LabirynthGen. 

Управление кубом на кнопки Q W E A S D.
